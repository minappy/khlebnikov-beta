# 🎭 Khlebnikov Poetry Generator

Генератор поэзии в стиле Велимира Хлебникова на основе нейросети GPT-2.

## 📋 О проекте

Бета-версия генератора, обученная на 100 стихотворениях Хлебникова. Модель дообучается на русскоязычной GPT-2 Small и способна генерировать тексты с элементами авторского стиля: окказионализмами, ритмическими повторами, звукописью.

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/YOUR-USERNAME/khlebnikov-beta.git
cd khlebnikov-beta
```

### 2. Создайте виртуальное окружение

```bash
python -m venv .venv
```

### 3. Активируйте окружение

Windows:

```bash
.\.venv\Scripts\Activate.ps1
```

Linux/Mac:

```bash
source .venv/bin/activate
```

### 4. Установите зависимости

```bash
pip install -r requirements.txt
```

### 5. Соберите корпус текстов

```bash
python corpus_builder.py
```

### 6. Обучите модель

```bash
python trainer.py
```

⏳ Займёт 20-40 минут на CPU.

### 7. Запустите генератор

```bash
python simple_app.py
```

Откройте браузер: http://127.0.0.1:8080

### 🎮 Использование

### ❓ Частые вопросы
Q: Почему модель генерирует не как Хлебников?
A: Это бета-версия. Увеличьте количество стихов в corpus_builder.py (строка poems[:100]) и эпохи в trainer.py (num_train_epochs=5).

Q: Как улучшить качество?
A: Соберите больше стихов и увеличьте число эпох обучения.

Q: Нужен ли интернет для генерации?
A: Нет, только при первом запуске для скачивания модели.
